# Rinne (Linux) Support

This repository is the public issue tracker for Rinne on Linux.

Website: https://rinne-btn.pages.dev/

## Supported environment
Rinne currently supports **Ubuntu 24.04 or later**.

Other Linux distributions may work, but they are not officially supported at this time.

## Use GitHub Issues
Please use **Issues** for:
- Bug reports
- Feature requests
- Compatibility reports, documentation fixes, and product feedback

## Crash reports (Linux)
If the app crashes, crash reports are typically written under:

- `~/.local/share/rinne/crash/`

Please include the newest `crash-gui-*.txt` file in your issue.
Please redact any sensitive paths, usernames, or personal information if needed.
